# levenshtein-rs

Messing around with Levenshtein distance.

**Given**

```
s1="prefer", s2="pretense"
```

**Result**

- We need to make a total of **4** changes to turn `s1` (prefer) into `s2` (pretense).
- We need to make the following 4 changes (Note: `Ignore` does not count as a change):

```
[
  ["Ignore", "p"], 
  ["Ignore", "r"], 
  ["Ignore", "e"], 
  ["Substitute", "f", "t"], 
  ["Ignore", "e"], 
  ["Substitute", "r", "n"], 
  ["Add", "s"], 
  ["Add", "e"]
]
```
